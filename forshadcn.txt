## Full Steps to Initialize shadcn/ui with JavaScript

1.  **Install Tailwind CSS**:
    Run `npm install tailwindcss @tailwindcss/vite` in your project. 

2.  **Initialize Tailwind**:
    Create a `tailwind.config.js` file with:
    ```js
    /** @type {import('tailwindcss').Config} */
    export default {
      content: ["./index.html", "./src/**/*.{js,jsx}"],
      theme: { extend: {} },
      plugins: [],
    }
    ```
    Add `@tailwind base; @tailwind components; @tailwind utilities;` to your `src/index.css`. 

3.  **Create `jsconfig.json`**:
    Set up path aliases by creating `jsconfig.json` in your root:
    ```json
    {
      "compilerOptions": {
        "baseUrl": ".",
        "paths": {
          "@/*": ["./src/*"]
        }
      }
    }
    ```

4.  **Update `vite.config.js`**:
    Configure Vite to recognize the `@` alias:
    ```js
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'
    import path from 'path'
    import tailwindcss from '@tailwindcss/vite'

    export default defineConfig({
      plugins: [react(), tailwindcss()],
      resolve: {
        alias: {
          "@": path.resolve(__dirname, "./src"),
        },
      },
    })
    ```

5.  **Initialize shadcn/ui**:
    Run `npx shadcn@latest init` and follow the prompts to select your style, color, and CSS variables. 

6.  **Disable TypeScript**:
    Open the generated `components.json` and set `"tsx": false` to ensure components are generated in JavaScript. 

7.  **Add Components**:
    Install any component using `npx shadcn@latest add <component-name>`, for example:
    ```bash
    npx shadcn@latest add button card input
    ```

8.  **Use Components**:
    Import and use components in your app:
    ```jsx
    import { Button } from "@/components/ui/button"

    export default function App() {
      return (
        <div className="p-6">
          <Button>Click Me</Button>
        </div>
      )
    }
    ```
